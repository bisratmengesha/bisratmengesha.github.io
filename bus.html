Bus Tracker MIT xPRO Exercise

This exercise uses the MBTA bus data in real-time to show the stops for bus route 1. Then, it adds an animated marker on the map to highlight the bus routes. Want to give it a try? You can check the code on Github You will need to create a public token from mapbox.com to access the map API and add your token in the mapanimation.Js file.
